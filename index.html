Download

这里先提供各系统下载入口占位（可随时替换为真实安装包与校验信息）。

多平台

macOS

适用于 Apple Silicon / Intel。建议通过签名安装包分发。

占位路径：`./downloads/Filmeto-mac.dmg`
下载 DMG（占位） macOS 12+

Windows

推荐提供安装版（.exe）与便携版（.zip）。

占位路径：`./downloads/Filmeto-win.exe`
下载 EXE（占位） Windows 10+

Linux

可提供 AppImage / deb / rpm，多发行版更友好。

占位路径：`./downloads/Filmeto-linux.AppImage`
下载 AppImage（占位） x86_64
